🛒 E-Commerce Website (React)


A frontend e-commerce web application built using React + Vite. Users can browse products, add items to cart, and manage cart globally using Context API.


🚀 Features


Product listing

Add to Cart

Remove from Cart

Quantity handling

Global Cart (Context API)

Responsive Navbar

Login & Register UI


🛠 Tech Stack


React.js

Vite

Context API

CSS

📦 Setup


git clone <your-repo-link>

cd frontend

npm install

npm run dev
